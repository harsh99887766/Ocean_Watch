## Project Overview
**ArgoNauts** is an interactive dashboard that visualizes oceanographic data and provides AI-driven summaries using a **Retrieval-Augmented Generation (RAG)** pipeline. Users can explore salinity and temperature trends in real-time and interact with an AI-powered chatbot for contextual insights.

---

## Key Features
- **Real-time Data Visualization:** Interactive charts and maps for salinity and temperature.  
- **AI Summaries:** Context-aware insights generated via **RAG**.  
- **Chatbot Integration:** Ask questions about oceanographic data and receive intelligent answers.  
- **Automation Pipeline:** Data extraction to visualization automated with **n8n**.  
- **Responsive UI:** Built with React.js for seamless desktop and mobile experience.

---

## Tech Stack
- **Frontend:** React.js  
- **AI / ML:** RAG, OpenAI API  
- **Visualization:** Recharts, Leaflet  
- **Automation / Workflows:** n8n  
- **Backend:** Node.js / Express (if applicable)
